## 개요
게임 '림버스 컴퍼니' 의 전투 시스템 구현 및 데이터 드리븐,

데이터 관리, 자동 세이브, 림버스 컴퍼니의 턴제 전투 시스템을 연습한 프로젝트입니다.


## 개발 환경
+ Unity 2022.3.16f1
+ Visual Studio 2022
+ Window 11


## 주요 목표
+ 게임 '림버스 컴퍼니' 구현
  + 합 & 턴제 전투 로직
  + 인격 편성
  + 자판기 (인격 & 에고 구매)
  <br />

+ 데이터 관리
  + 어드레서블을 사용한 에셋 캐싱, 언로드
  + 데이터 드리븐을 통한 전투 맵 & 몬스터 & 시스템 배치
  <br />

+ 데이터 세이브 & 로드
  + Json 기반 세이브 & 로드 시스템 구현 
  + 특정 조건 (인격 & 에고 구매, 편성, 재화 소모) 시 자동 세이브 기능
  + 데이터 무결성 체크 및 문제 발견 시 신규 데이터 작성 기능
  <br />

+ 디자인 패턴을 활용한 버프 & 디버프 시스템
  + 전략 패턴 기반 버프 & 디버프의 호출부와 구현부 분리
  + 컴포지트 패턴 구조를 통한 버프 조합 및 관리
  + Buff_Base -> Buff_이름 형태의 상속 구조 기반 세부 버프 & 디버프
  <br />
